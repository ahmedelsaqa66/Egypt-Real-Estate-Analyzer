## 🇪🇬  - نظرة عامة على المشروع

هذا المشروع هو حل تقني متكامل لتحليل سوق العقارات في مصر (التجمع الخامس والشيخ زايد). يقوم البرنامج بسحب البيانات آلياً من مواقع العقارات، تنظيفها، وعرضها في لوحة تحكم تفاعلية تساعد المستثمرين والمشترين على اتخاذ قرارات مبنية على أرقام حقيقية.

### 🚀 المميزات الرئيسية:
* **سحب البيانات (Web Scraping):** سكربت تلقائي لجمع أحدث إعلانات الشقق والأسعار.
* **لوحة تحكم تفاعلية (Dashboard):** مبنية باستخدام Streamlit تتيح الفلترة حسب (السعر، المساحة، والمنطقة).
* **تحليل بصري (Visualizations):** رسوم بيانية توضح متوسط سعر المتر واتجاهات السوق.
* **تصدير البيانات:** إمكانية تحميل النتائج المفلترة في ملف Excel أو CSV.

### 🛠️ الأدوات المستخدمة:
* **لغة البرمجة:** Python.
* **المكتبات:** Pandas, Plotly, Streamlit.
* **هيكلة المشروع:** تنظيم احترافي للمجلدات لسهولة التعديل والتطوير.

---

## 🇬🇧  - Project Overview

This is a professional data solution for the Egyptian real estate market. It automatically scrapes data from top listings, cleans it, and provides an interactive dashboard for buyers to compare prices between **New Cairo** and **Sheikh Zayed**.

### 🚀 Key Features:
* **Automated Web Scraping:** Gathering real-time property listings.
* **Streamlit Dashboard:** Interactive filters for (Price, Area, and Region).
* **Advanced Visualizations:** Charts identifying market trends and price per meter.
* **Data Export:** Filtered results can be exported for further analysis.

### 🛠️ Tools & Technologies:
* **Python** (Pandas, Plotly, Streamlit).
* **Web Scraping** (BeautifulSoup/Requests).
* **Project Structure:** Professional directory organization.
